记录一下自己的洛谷做题，顺便展示一下最近学的欧拉筛！

```
c++
#include<bits/stdc++.h>
using namespace std;

int t,n,prime[100001]={0};
bool all[100001]={0};

int main() {
    cin>>n;

    for(int i=2;i<n;i++) {
        if(all[i]==0) {
            prime[t++]=i;
        }

        for(int j=0;j<t&&i*prime[j]<=n;j++) {
            all[i*prime[j]]=1;
            if(i%prime[j]==0)
                break;
        }
    }
    for(int i=0;i<t;i++) {
        cout<<prime[i]<<" ";
    }
}




```

欧拉筛的基本原理就是：

每个素数的倍数都是合数，而每个合数可以表示为一个最小的素数和另一个系数的乘积。

如12==3\*4
    12==2\*6
    12==3\*4 
    12==6\*2

因此我们只用筛2\*6筛，大幅减少时间，能够达到o(n)水平。
